어떻게 10,000개를 막나요?

바뀌지 않는 가운데 번호를 등록합니다

070-1234-0000 ~ 9999

예를 들어 070-1234를 일괄 차단하면 그 뒤에 붙는 네 자리 조합 10,000개가 차단 대상이 됩니다. 같은 업체가 끝자리만 바꿔 거는 패턴에 맞춘 방식입니다.

중요: 070 번호 전체를 막는 기능은 아닙니다. 실제로 반복 수신한 특정 가운데 번호 대역만 골라 등록하세요. 연락처 보호를 켜면 저장된 번호는 일괄 차단 대상에서 자동으로 제외됩니다.

070 번호를 전부 차단하나요?

아니요. 사용자가 등록한 특정 가운데 번호 대역만 차단합니다. 필요한 전화를 놓치지 않도록 070 전체 차단은 제공하지 않습니다.

차단된 전화는 어디에서 확인하나요?

iPhone 통화 기록에서 스팸독: 차단함 표시를 확인할 수 있습니다. iOS 구조상 앱 자체는 사용자의 수신 기록에 접근할 수 없습니다.

정말 광고와 결제가 없나요?

네. 전화번호 하나 차단, 가운데 번호 패턴 차단, 패턴 80개와 최대 80만 번호 처리는 광고와 결제 없이 사용할 수 있습니다.

문자도 차단하나요?

현재 버전은 iPhone 전화 수신 차단과 발신자 표시에 집중합니다. 문자 차단 기능은 아직 제공하지 않습니다.

아이폰 070 차단 방법을 먼저 보고 싶나요?

설정 위치, 가운데 번호를 고르는 법, 차단됐는데 음성사서함이 남는 이유까지 정리했습니다.
