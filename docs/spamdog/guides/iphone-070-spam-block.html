아이폰 스팸전화 차단 가이드

끝자리만 바뀌는 070 전화, 하나씩 차단하지 마세요

같은 가운데 번호를 쓰면서 뒤 네 자리만 바꿔 오는 스팸이라면, 반복되는 대역 하나를 등록해 10,000개를 한 번에 차단할 수 있습니다.

작성·업데이트: 2026년 7월 21일

30초 요약 반복되는 070 전화의 가운데 네 자리를 확인하고, 스팸독에 해당 대역을 등록한 뒤 iOS의 통화 차단 설정을 켜면 됩니다. 070 전체가 아니라 직접 고른 대역만 차단하는 방식입니다.

왜 한 번호씩 차단해도 계속 걸려올까요?

발신 측이 한 대역 안에서 뒤 네 자리만 바꾸면, 보이는 전화번호는 매번 달라도 가운데 번호는 그대로일 수 있습니다. 이때 완전한 번호 하나만 차단하면 다음 번호는 다시 울립니다.

070-1234-0182

070-1234-6741

070-1234-9903

끝 네 자리는 달라도 가운데 1234는 같습니다.

가운데 번호 070-1234를 패턴으로 등록하면 070-1234-0000부터 070-1234-9999까지, 총 10,000개 조합이 차단 대상이 됩니다.

아이폰에서 070 스팸전화 차단하는 순서

  1. 최근 통화에서 반복되는 번호를 확인하세요.
    서로 다른 전화의 가운데 네 자리가 같은지 비교합니다.
  2. 스팸독에 전화번호를 입력하거나 붙여넣으세요.
    완전한 번호를 넣으면 앱이 가운데 번호 일괄 차단을 제안합니다.
  3. ‘10,000개 일괄 차단’을 선택해 밀봉하세요.
    연락처 보호를 켜 두면 저장된 번호는 일괄 차단에서 제외됩니다.
  4. iOS 설정에서 스팸독을 활성화하세요.
    설정 → 앱 → 전화 → 통화 차단 및 발신자 확인에서 스팸독을 켭니다.
070 전체 차단과는 다릅니다. 070 인터넷전화에는 정상적인 병원, 배달, 회사 전화도 있을 수 있습니다. 실제로 스팸이 반복된 특정 가운데 번호만 등록하는 편이 안전합니다.

차단이 작동했는지 확인하는 법

차단 대상 전화는 벨과 수신 알림 없이 음성사서함으로 넘어갈 수 있습니다. iPhone 통화 기록에 스팸독: 차단함이 표시됐다면 정상적으로 작동한 것입니다.

iOS의 Call Directory 구조상 스팸독 앱은 사용자의 실제 수신 기록이나 통화 내용을 볼 수 없습니다. 그래서 앱 안에 ‘몇 통 차단’ 통계를 만드는 대신, iPhone 통화 기록의 표시로 확인합니다.

자주 묻는 질문

아이폰에서 070 번호 전체를 차단할 수 있나요?

필요한 전화까지 막힐 수 있어 070 전체 차단은 권하지 않습니다. 스팸독은 사용자가 실제로 반복 수신한 특정 가운데 번호 대역만 골라 차단합니다.

차단된 전화가 음성사서함에 남는 것은 정상인가요?

네. iPhone에서는 차단된 발신자도 음성사서함을 남길 수 있습니다. 벨과 수신 알림이 없고 통화 기록에 스팸독: 차단함이 표시되면 차단이 작동한 것입니다.

가운데 번호 하나로 왜 10,000개가 차단되나요?

뒤 네 자리는 0000부터 9999까지 10,000가지입니다. 가운데 번호를 고정하고 이 조합 전체를 차단 대상으로 등록하기 때문입니다.

문자 스팸도 차단되나요?

아니요. 현재 스팸독은 전화 수신 차단과 발신자 표시에 집중하며 문자 차단 기능은 제공하지 않습니다.

반복되는 번호를 찾았다면 바로 막아보세요

가운데 번호 패턴 80개, 최대 80만 번호까지 광고와 결제 없이 사용할 수 있습니다.
