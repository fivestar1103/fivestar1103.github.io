1. AIDA (Axie Infinity 데이터 분석 서비스) (2023.03.01. ~ 2023.05.31.)

  • Web3 게임 Axie Infinity: Originis의 사내 프로 선수를 지원하기 위한 데이터 기반 분석 서비스를 백엔드부터 프론트엔드, 배포까지 총괄하여 개발했습니다.
  • 공식 API를 활용하여 게임 대전 기록을 수집/분석하고, 덱 빌딩 개선 및 브리딩(아이템 제작) 효율화를 위한 유의미한 인사이트를 제공하는 백엔드 시스템을 구축했습니다.
  • Python Flask와 MySQL을 사용하여 안정적인 웹 서비스 및 데이터베이스를 설계했으며, Google Cloud Platform(GCP) 환경에 성공적으로 배포하여 실시간 서비스 운영 경험을 쌓았습니다.

2. Emoji Swipe Rescue (1인 개발 및 출시) (2025.04.23. ~ 2025.05.06.)

샘플 코드
  • 기획부터 개발, 디자인, 앱스토어 출시 및 운영까지 서비스 전체 라이프사이클을 1인 개발로 경험한 프로젝트입니다.
  • Google AdMob SDK를 연동하여 광고 수익화 모델을 구현했으며, Game Center 연동으로 리더보드 기능을 제공하는 등 외부 서비스와의 통합 경험을 보유하고 있습니다.
  • Unity 기반의 게임 클라이언트 개발 역량과 더불어 i18n(영어, 한국어) 지원을 통해 글로벌 서비스를 위한 기반을 마련했습니다.
  • 게임 시작 시 로딩 시간 단축을 위해 비동기 씬 로딩 시스템을 구현하여 사용자 경험(UX)을 개선했습니다.

4. 필연과 우연 (데이터 관리 시스템 설계) (2024.03.10. ~ 현재)

샘플 코드
  • 개발 리드로서 프로젝트를 총괄하며, 특히 게임 데이터 관리 시스템을 설계하고 구축했습니다.
  • Google Sheets를 외부 데이터베이스처럼 활용하고 CSV 파싱을 통해 게임에 동적으로 데이터를 적용하는 이벤트 기반 아키텍처를 구현했습니다. 또한, 수정 시 매번 시트를 직접 다운로드할 필요가 없도록 유니티 에디터 기능을 구현하여 개발 편의성을 높였습니다.
  • 이를 통해 기획자가 직접 데이터를 수정하고 게임에 즉시 반영할 수 있도록 만들어, 기획자와 개발자의 협업 효율을 크게 개선하고 운영 편의성을 높였습니다.

5. 감정 인식 AI를 활용한 절차적 생성 퍼즐 어드벤처 게임 (2024.09.10. ~ 2024.12.10.)

샘플 코드
  • 동적인 게임 환경을 생성하기 위한 절차적 생성(PCG, Procedural Content Generation) 알고리즘을 설계 및 구현함.
  • 복잡한 PCG 던전을 구성하기 위해 셀룰러 오토마타, 들로네 삼각분할, MST(최소 신장 트리) 및 A* 알고리즘을 활용함.
  • 음성-텍스트 변환(STT) 및 감정 인식(SER) 기술을 게임 내부에 통합함.
  • 팀을 이끌고 프로젝트 병합 및 버전 관리를 총괄하였음.
